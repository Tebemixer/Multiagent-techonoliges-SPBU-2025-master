# Multiagent-techonoliges-SPBU-2025-master

Репозиторий переносит исходный проект на чистый Python: вместо SPADE здесь используется синхронное моделирование обмена сообщениями между агентами. Это упрощает запуск и даёт возможность экспериментировать с алгоритмом без установки XMPP‑сервера.

## Требования

- Python 3.10+
- (опционально) `pip install -r requirments.txt` — файл оставлен для совместимости и сообщает, что внешних зависимостей нет.

## Как запускать

1. **Простая система** (`dummy_system.py`) — агенты посылают одно число центру, который считает среднее:

   ```bash
   python dummy_system.py
   ```

2. **Система с ограниченной связностью** (`limited_communication_system.py`) — имитация рассылки знаний по графу и подсчёт стоимости коммуникаций:

   ```bash
   python limited_communication_system.py
   ```

`matrix_generator.py` отвечает за генерацию матрицы смежности (по умолчанию — линейная цепочка). Измените функцию, чтобы проверить алгоритм на другом графе.

## Описание алгоритма
Описание требует доработки, как и код, не реализовано распространение сигнала МОЛЧАТЬ.
### Постулаты алгоритма
#### 1. С каждой n итерацией вершина графа узнает о других вершинах на расстоянии n от нее.
#### 2. Если на этой итерации вершина не получила новых знаний, то вершина знает о всех вершинах графа, и отправляет итоговые данные в центр. Соседним вершинам отправляется сигнал МОЛЧАТЬ.
#### 3. После получение сигнала МОЛЧАТЬ вершина обязана передать этот сигнал своим соседям.
