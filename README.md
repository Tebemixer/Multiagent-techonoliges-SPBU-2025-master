## Установка зависимостей

```bash
pip install -r requirments.txt
git clone https://github.com/javipalanca/spade.git
cd spade
python setup.py install
cd ..
```

## Запуск сервера

В отдельном терминале:

```bash
spade run
```

## Запуск программы

В отдельном терминале:

```bash
python limited_communication_system.py
```
