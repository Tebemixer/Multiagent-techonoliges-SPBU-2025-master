# Multiagent-techonoliges-SPBU-2025-master

Репозиторий переносит исходный проект на чистый Python: вместо SPADE здесь используется синхронное моделирование обмена сообщениями между агентами. Это упрощает запуск и даёт возможность экспериментировать с алгоритмом без установки XMPP‑сервера.

## Требования

- Python 3.10+
- (опционально) `pip install -r requirments.txt` — файл оставлен для совместимости и сообщает, что внешних зависимостей нет.

## Как запускать

1. **Простая система** (`dummy_system.py`) — агенты посылают одно число центру, который считает среднее:

   ```bash
   python dummy_system.py
   ```

2. **Система с ограниченной связностью** (`limited_communication_system.py`) — имитация рассылки знаний по графу и подсчёт стоимости коммуникаций. По умолчанию сценарий запускается для пяти агентов (`NUM_AGENTS = 5` внутри файла), что совпадает с расчётами из задания:

   ```bash
   python limited_communication_system.py
   ```

`matrix_generator.py` отвечает за генерацию матрицы смежности (по умолчанию — линейная цепочка). Измените функцию, чтобы проверить алгоритм на другом графе.

## Описание алгоритма

Алгоритм имеет постулаты:

1. Каждая итерация делится на три фазы: агент отправляет всем соседям **весь** текущий набор знаний (одно сообщение на соседа), затем принимает такие же сообщения от соседей и обновляет локальный словарь. Благодаря последовательности «отправка → получение → размышление» знание распространяется послойно на расстояние `n` за `n` итераций, при этом стоимость сообщения не зависит от числа чисел внутри.
2. Если в фазу обновления агент не получил ничего нового, он помечает себя как «готового к отчёту». На следующей итерации такой агент отправляет среднее в центр (стоимость `1000`) и параллельно рассылает соседям сигнал «МОЛЧАТЬ» (каждый сигнал стоит `10`), после чего окончательно выходит из игры.
3. Любой агент, получивший сигнал «МОЛЧАТЬ», немедленно пересылает его всем соседям, кроме отправителя, и тоже переходит в молчащее состояние — так реализуется лавинообразное затухание сети.

С таким поэтапным обменом результаты совпадают с ожидаемыми: цепочка из трёх агентов стоит `1120` условных единиц, а из пяти — `1360` (три итерации обмена знаниями и одна итерация с отчётом и сигналом молчания).
